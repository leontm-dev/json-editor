# Welcome to simple-json-editor

This package is a easy package to use for editing json files. This package has a hell load of functions and parameters to modify your json files as you want.

## Installation

1. Insert `npm install --save simple-json-editor` into your folder's terminal
2. Require the package with `require("simple-json-editor")` inside your JavaScript file.
3. Start using the editor. You will find the docs down below
